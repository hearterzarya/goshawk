# Goshawk Logistics

Premium logistics brokerage website built with Next.js 14, TypeScript, Tailwind CSS, and GSAP.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- GSAP + ScrollTrigger
- React Hook Form + Zod
- Framer Motion (micro interactions)

## Project Structure

- `/app` - Next.js App Router pages
- `/components` - Reusable UI components
- `/hooks` - Custom React hooks (GSAP, etc.)
- `/lib` - Utilities and helpers
- `/data` - Static data (services, testimonials, FAQs)
- `/styles` - Global styles
